
# RPA 学院 - 学习门户 产品需求文档 (PRD)

## 1. 项目概述
RPA 学院是一个面向 RPA 开发者（特别是 UiBot 用户）的专业在线学习平台。该平台采用全动态数据驱动架构，所有教学分类和视频内容均通过后端接口（当前为 Mock 服务）实时获取。

## 2. 核心功能与页面描述

### 2.1 动态课程展示
- **异步加载**: 页面初始化时异步获取分类列表，点击分类时异步获取课程列表。
- **视觉反馈**: 在数据传输期间，系统必须展示骨架屏 (Skeleton Screen) 以减少用户的感知等待时间，提供极致的丝滑感。
- **全动态卡片**: 课程卡片内的分类标签（如 "基本命令"）必须根据当前接口返回的分类名称动态生成。

### 2.2 智能助教 (AI Tutor)
- **实时响应**: 集成 Google Gemini 3 系列模型，提供秒级反馈。
- **上下文理解**: 模拟真实的助教环境，解答 UiBot 命令使用和流程排障问题。

## 3. 技术设计准则
- **状态即时重置**: 在切换分类时，必须立即清空当前列表并触发 Loading 状态，模拟真实的远程 API 调用反馈。
- **接口解耦**: 所有的 UI 组件不直接硬编码数据，而是通过 `courseService` 进行中转。

## 4. 非功能性需求
- **性能**: 模拟 400-600ms 的网络延迟，测试骨架屏的平滑过渡。
- **一致性**: 确保侧边栏选中的 `activeCategory` 与右侧渲染的课程 `category` 属性完全匹配。

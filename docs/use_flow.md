
# 页面操作流程描述 (Use Flow)

## 1. 场景：动态分类与课程加载
**描述**：用户通过点击侧边栏实现右侧内容的实时动态更新。

### 操作步骤
1. **系统初始化**: 页面加载，左侧导航栏显示 6 个骨架条。
2. **分类呈现**: 600ms 后，分类列表出现，默认选中“基本命令”。
3. **内容预加载**: 右侧课程区域在分类选中的瞬间进入 Loading 态，展示 6 个课程占位骨架。
4. **内容渲染**: 400ms 后，右侧课程卡片滑入显示，分类标签显示对应的动态名称。
5. **切换交互**: 用户点击“Mage AI”。
   - **瞬间动作**: 右侧旧内容立即消失，重新显示骨架占位图。
   - **完成动作**: 新分类下的课程数据加载并渲染。

### 关键元素 XPath (用于自动化测试)
- **正在加载的骨架屏**: `//div[contains(@class, 'animate-pulse')]`
- **动态生成的课程标签**: `//span[contains(@class, 'bg-white/10')]`
- **空状态提醒**: `//p[text()='该分类下暂无视频课程']`
